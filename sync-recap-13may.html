SPECS TÉCNICAS.

Definiciones cerradas hoy. Base para el brief a Camel y el trabajo de modelado.

Polígonos por personaje
10k – 30k

Alta calidad visual sin afectar performance. Jorge confirma el tope exacto antes de dar el brief.

Tipo de rig
Humanoid
+ dedos

Rig avanzado para Unity. Cabeza con movimiento independiente para mejorar animaciones.

Accesorios
Modulares

Crestas, peinados y extras se activan / desactivan desde el rig. El modelo base queda limpio.

Sistema de agarre
Mano cerrada

El objeto se ancla a la mano cerrada. Sin sistema complejo de grip por objeto.

Interior del avión
Pasillo amplio

Pasillo central más ancho. Espacios de jugabilidad adicionales. Basado en el Lockheed Constellation.

Elementos destructibles
3 aberturas

3 puntos de ruptura predefinidos por lado del avión. Elementos removibles para las mecánicas de caos.
